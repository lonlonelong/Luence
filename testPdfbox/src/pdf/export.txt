大一上学期，当时觉得寝室里很吵，有的打游戏，有的打电话，而我又喜
欢安静的敲会儿 C 代码，看一些感兴趣的文章和新闻，不得已只能天天戴耳机
边听歌边做自己的事。这是我 当初 为了加团队的主要原因，目的就 只有一个，
就是为了找一个安静的环境，一个凳子和一张桌子。  
后面 过了 两 个月，通知 进了团队，出于对团队的好奇以及内心的兴奋， 只
要 一有时间就去团队呆 。 运气比较好还有空位可以坐， 一些 和我一起 选进 团
队，但是比我晚几天才去团队的就找不到位置坐了，也是一部分原因他们很少
到团队 学习 。 更多的时间呆在团队，心里也有些暗暗得意的样子，哈哈，肯定
能比 他们 学到更多的东西。后面 从网上， 和 老师那里，渐渐了解到了更多关于
计算机的，才发现自己有很多不知道但是感兴趣的东西。 然后 就去搜资料了
解，学习，开始了一个广度学习的模式。  
到大一下学期的三月份，都是以这样的一个思想去学习，期间按照 C#从入
门到精通这本书学了 C#基础用法，跟着另一本 C#实例的书做了一些简单的，不
涉及联网的 GUI 桌面应用，跟着网易云课堂学了 Python 基础，做了一个类似
2048 的桌面游戏，后面又去网上找资料，找到了一个用 Python 开发游戏的一
系列文章教程，出于兴趣，天天学的还挺有激情的。后面又学了 html， CSS 和
JavaScript，当时只是粗略的能够读懂一些简单网页源码，但是还不会自己
写。  
现在想来 ，当时学的挺杂的，虽然看起来对我现在所学的没有什么用处，
但我 丝毫 没有觉得那段日子 过 的不值，完全没有后悔应该把那段时间用去打游
戏 或者 其他 娱乐。我从那段时间获得的是更多对于计算机的兴趣和激情，更早
的掌握了适合自己的学习方法，也改变了我对以后的人生路的看法。  
到大一 下 学期那天分方向，我也不太明确自己还选什么，因为对各个方向
都模糊，不知道是干嘛的，索性做个排除法。嵌入式这个硬件我不太喜欢弄，
安卓这个东西看上去挺好的，但我觉得如果有一门技术可 以开发安卓和 iOS，
那我肯定就选它了，就只剩下 web 方向，后面发现之前学的 html， JavaScript
可以派上用场，又是一阵暗暗自喜，哈哈。既然选择了方向，我就相信自己是
能够做到最好的。因为我觉得时间花的多，就不会太差。  
从此开启深度学习模式。为了给自己赢得一个更高的起点，天天给自己定
一个计划， 若 第二天不完成 ， 就会心里面抽自己两巴掌，说实话 当时 自己给自
己的压力还 真 不小。大一即将结束，但是早已没有那种对假期的渴望，应该是
被自己定的任务压昏了头脑，决定暑假在学校好好学习，等有回家的欲望了再
回去放松一下。  
有一天梁老师给了 我一个小项目题目让我用把程序写出来，虽然没有做
过，但我习惯查资料，然后谷歌了一会儿，倒腾了三个小时实现了功能。后面
给了我一个公司里的后台业务系统练手，是我第一个真正意义上的项目，遇到
了很多麻烦，花费了很多精力和时间，但是同等收获了很多，让我更加清楚
web 方向该干的，熟悉了那些基础的东西，也指出了哪些东西是自己以后该学
的，受益匪浅。后面又做了团队的新生报名小系统，团队官网和天天用的签到
系统，进一步巩固了所学的，更加熟悉了那些简单的套路。后面越来越觉得自
己能做的都只是一些简单的东西，于是又回到了学习状态给自己充 电。学了一
个月后，就是前两周，觉得自己应该检验检验学习的成效，就去网上查了查资
料，发现一个类似京东的电商网站挺适合的，又开始码 这个 项目的 代码，一直
到现在。  
说说我对团队的看法，从团队的积极性来看，每个人都是愿意为团队出力
的，比如团队的搬迁和灵动物联网大赛的举办；从纪律来看，最好的是大一，
像小老鼠一样， 更 应该约束的是大二大三的部分，说话声音大， 好像 觉得自己
说话 不会吵到别人，讨论也 不 找一个更好的场合；从学习来看，大一应该也是
最顺利的了，一切都在组长们的掌握之中，但不好的也大大的存在，他们没有
更多的时间去探索自己感兴趣的，少了些 学习自由，多了约束和压力。大二的
经过开放性实验后，应该都有了感受和醒悟，需要去调整和坚持。大三的不说
其他的， 55 个小时就说明一切了，哈哈 ！  
我 认为 下一届 管理 团队更适合一个小团队合作管理，比如日常事务组织一
个人，纪律一个人，学习一个人，每个人做自己擅长的，这样效率高而且都比
较轻松。我就比较适合规划一下学习和项目方面的，让我去组织会议的话，我
觉得我把自己想说的话三五分钟说完就完事儿了，不太喜欢说一些套路话，可
能是我现在 还没 到那个角色，没经历过 就 不习惯吧，让我去管纪律，我没法一
下子严肃起来，哈哈，不是个严肃的人，天性爱笑。这样也没必要轮着 来做负
责人，管理团队带得好的话，也不必再选一个最终负责人，到时管理班子肯定
会有一个说话好使的人。  
对于 团队未来的规划，第一点，最基本的，我希望团队里的人都是有实力
的 ； 第二点，希望能够有更大的地方，让更多有理想的人在 这里学习 ； 第三
点，希望能够找到外包，给团队输血，让团队拥有更好的条件和环境，也能更
直接的激发大家的学习兴趣。
